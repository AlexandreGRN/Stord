Home

Stord is a stock management application that helps you manage your stock and keep track of your sales. It is a simple and easy to use application that can be used by anyone. It is available on both android and web.

The main goal of the application is to simplify stock manager work by overseeing more frequently inventories in real time. It can be used as a way to control more wastes thanks to graphs accessible via the Stord web page.

Start now by creating your account and by downloading the Stord android application.
